How to pair
  1. Find the physical Hue Bridge (small white square plugged into the router).
  2. Press the round button on top.
  3. Within 30 seconds, run on studiopi: 
    ssh admin@studiopi.local
cd /home/admin/screen-mgr
python3 scripts/hue_pair.py
  4. Reload this page. The Lights tab should populate.
